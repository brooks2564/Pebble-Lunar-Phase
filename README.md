# Lunar Phase

A Pebble watchface that tracks the moon.

## Features

- **Live moon phase** — accurately drawn crescent, gibbous, quarter, full, and new moon updated daily
- **Phase name** — displays the current phase (e.g. Waxing Crescent, Full Moon)
- **Current weather** — temperature and conditions pulled from your location
- **Sunrise & sunset** — rise and set times shown at the bottom
- **Starfield background** — scattered stars that never overlap the moon
- **Shooting star** — flick your wrist to send a shooting star across the screen

## Platforms

Pebble Time 2 (emery) · Pebble Time Round 2 (gabbro)
